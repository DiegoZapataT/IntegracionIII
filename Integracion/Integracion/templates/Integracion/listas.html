Listas

En Python existen varios tipos de datos compuestros y que tienen secuencias definidas, uno de estos tipos de datos son las listas.

Entre los valores con secuencia el mas versatil de todos es la lista, para definir una se debe crear una variable de la manera a = [], donde los corchetes indican que es un objeto de tipo lista que en ese momento se encuentra vacio. Por Ejemplo: 

                  >>> Boleta = ['pan', 'huevos', 100, 1234]
                  >>> Boleta
                ['pan', 'huevos', 100, 1234]

En el ejemplo anterior tenemos una lista que se crea con datos asignados de manera inmediata, podemos ver que contiene 4 valores, pero que para el caso de las listas cada una tiene un valor posicional, siendo el 0 para el primer valor y luego continua de manera ascendente.

Las listas en python son Heterogeneas, es decir, pueden estar conformadas por elementos de distinto tipo, incluidas mas listas dentro de la misma, y mutables, es decir sus elementos pueden modificarse. En otras palabras, una lista es una estructura de datos formada por una secuencia ordenada de objetos.
Los elementos de una lista pueden accederse mediante su indice, siendo 0 el indice del primer elemento: 

                  >>> boleta[0]
                  'pan'
                  >>> boleta[3]
                  1234


La funcion len() devuelve la longitud de la lista (cantidad de elementos):

                  >>> len(Boleta)
                  4

Los indices de una lista inician desde 0 hasta el tamaño de la lista -1, es decir si tenemos una lista ordenada de 5 elementos, el indice ira desde el 0 al 4. Tambien se puede unsar indices negativos, siendo -1 el indice del ultimo elemento: 

                  >>> Boleta[-1]
                  1234

Los indices negativos van desde -1 hasta -len(boleta), es decir desde -1 al -4.

A traves de los indices, pueden cambiarse los elementos de una lista en un lugar particular: 

                  >>> Boleta[1] = "carne"
                  >>> Boleta
                  ['pan', 'carne', 100, 1234

Metodos

Para trabajar con listas tenemos variados metodos para hacer el trabajo mas facil y rapido, estos son:

I.- append()

Este metodo agrega un elemento al final de una lista: 

                  >>> versiones_plone = [2.5, 3.6, 4, 5]
                  >>> print versiones_plone
                  [2.5, 3.6, 4, 5]
                  >>> versiones_plone.append(6)
                  >>> print versiones_plone
                  [2.5, 3.6, 4, 5, 6]

II.- count()

Este metodo recibe un elemento como argumento, y cuenta la cantidad de veces que aparece en la lista:

                  >>> versiones_plone = [2.1, 2.5, 3.6, 4, 5, 6]
                  >>> print "6 ->", versiones_plone.count(6)
                  6 -> 1
                  >>> print "5 ->", versiones_plone.count(5)
                  5 -> 1
                  >>> print "2.5 ->", versiones_plone.count(2.5)
                  2.5 -> 1

III.- extend()

Este metodo extiende una lista agregando un iterable al final:

                  >>> versiones_plone = [2.1, 2.5, 3.6]
                  >>> print versiones_plone
                  [2.1, 2.5, 3.6]
                  >>> versiones_plone.extend([4])
                  >>> print versiones_plone
                  [2.1, 2.5, 3.6, 4]
                  >>> versiones_plone.extend(range(5,7))
                  >>> print versiones_plone
                  [2.1, 2.5, 3.6, 4, 5, 6]

IV.- index()

Este metodo recibe un elemento como argumento, y devuelve el indice de su primera aparicion en la lista:

                  >>> versiones_plone = [2.1, 2.5, 3.6, 4, 5, 6, 4]
                  >>> print versiones_plone.index(4)
                  3

El metodo a su vez admite como argumento adicional un indice desde el cual partir la busqueda, y opcionalmente tambien el indice hasta cual revisar (indice final): 

                  >>> versiones_plone = [2.1, 2.5, 3.6, 4, 5, 6, 4]
                  >>> versiones_plone[2]
                  3.6
                  >>> print versiones_plone.index(4, 2)
                  3
                  >>> versiones_plone[3]
                  4
                  >>> print versiones_plone.index(4, 5)
                  6
                  >>> versiones_plone[6]
                  4

Se debe saber tambien que si se le pasa un valor inexistente en la lista para realizar la busqueda, el metodo nos devolvera un error.

V.- insert()

Este metodo inserta un elemento x en la lista, en un indice i a asignar:

                  >>> versiones_plone = [2.1, 2.5, 3.6, 4, 5, 6]
                  >>> print versiones_plone
                  [2.1, 2.5, 3.6, 4, 5, 6]
                  >>> versiones_plone.insert(2, 3.7)
                  >>> print versiones_plone
                  [2.1, 2.5, 3.7, 3.6, 4, 5, 6]

VI.- pop()

Este metodo devuelve el ultimo elemento de una lista, y ademas lo borra de la misma:

                  >>> versiones_plone = [2.1, 2.5, 3.6, 4, 5, 6]
                  >>> print versiones_plone.pop()
                  6
                  >>> print versiones_plone
                  [2.1, 2.5, 3.6, 4, 5]

A su vez tambien se le puede asignar la posicion del valor que se desea mostrar y retirar de la lista.

VII.- remove()

Este metodo recibe como argumento un elemento, y borra la primera posicion donde lo encuentre en la lista:

                  >>> versiones_plone = [2.1, 2.5, 3.6, 4, 5, 6]
                  >>> print versiones_plone
                  [2.1, 2.5, 3.6, 4, 5, 6]
                  >>> versiones_plone.remove(2.5)
                  >>> print versiones_plone
                  [2.1, 3.6, 4, 5, 6]

El metodo devolvera un error si el numero que se desea eliminar no existe en la lista.

VIII.- reverse()

Este metodo invierte el orden de los elementos de una lista:

                  >>> versiones_plone = [2.1, 2.5, 3.6, 4, 5, 6]
                  >>> print versiones_plone
                  [2.1, 2.5, 3.6, 4, 5, 6]
                  >>> versiones_plone.reverse()
                  >>> print versiones_plone
                  [6, 5, 4, 3.6, 2.5, 2.1]

IX.- sort()

Este metodo ordena los elementos de una lista de menor a mayor:

                  >>> versiones_plone = [4, 2.5, 5, 3.6, 2.1, 6]
                  >>> print versiones_plone
                  [4, 2.5, 5, 3.6, 2.1, 6]
                  >>> versiones_plone.sort()
                  >>> print versiones_plone
                  [2.1, 2.5, 3.6, 4, 5, 6]

El metodo sort tambien admite la opcion reverse, por defecto, esta opcion se encuentra en False. Si se desea se ingresa sort(reverse=True) y ordenara de mayor a menor.

Recorrer una lista

Para recorrer o iterar de manera correcta un arreglo se debe recurrir a un ciclo for, como podemos ver en el siguiente ejemplo mezclando un ciclo for para recorrer un string y un if para comparar las vocales que existen en ese string: 

                  >>> vocales = 'aeiou'
                  >>> for letra in 'hermosa':
                  ...     if letra in vocales:
                  ...         print letra,
                  e o a
