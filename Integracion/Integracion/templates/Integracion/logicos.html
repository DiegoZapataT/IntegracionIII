Operadores Logicos

Estos son los operadores con los que se puede trebajar con valores booleanos que se usen durante nuestro codigo de programacion

Operador and

El operador and evalua si el valor del lado izquierdo y del lado derecho se cumple:

                        >>> True and False
                        False

Operador or

El operador or evalua si el valor del lado izquierdo o del lado derecho se cumplen:

                        >>> True or False
                        True

Operador not

El operador not devuelve el valor opuesto al valor booleano:

                        >>> not True
                        False

Si la expresion es True el valor devuelto es False, de lo contrario si la expresion es False el valor devuelto sera True. 

                        >>> not False
                        True

Ejemplos

A continuacion mostrare algunos ejemplos de operadores logicos:

Lo primero que realizamos es definir las variables a usar en los ejemplos:

                        a, b = 10, 20

Ejemplo para operador logico and:

                    if (a and b):
                        print "Las variables 'a' y 'b' son VERDADERO."
                    else:
                        print "O bien la variable 'a' no es VERDADERO " + \
                        "o la variable 'b' no es VERDADERO."

Ejemplo de operador logico or:

                    if (a or b):
                        print "O bien la variable 'a' es VERDADERA " + \
                        "o la variable 'b' es VERDADERA " + \
                        "o ambas variables son VERDADERAS."
                    else:
                        print "Ni la variable 'a' es VERDADERA ni " + \
                        "la variable 'b' es VERDADERA."

Ejemplo de operador logico not:

                    if not(a and b):
                        print "Ni la variable 'a' NO es VERDADERA " + \
                        "o la variable 'b' NO es VERDADERA."
                    else:
                        print "Las variables 'a' y 'b' son VERDADERAS."
