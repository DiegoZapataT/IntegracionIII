Tuplas

Las tuplas son objetos de tipo secuencia, un tipo de dato lista pero que es inmutable. esta a diferencia de las listas no se puede modificar de ningun modo a menos que se transforme a lista mediante el metodo list().

Metodos

Son muy similares a las listas y comparten varias de sus funciones y metodos integrados, aunque su principal difernecia es que son inmutables. El objeto de tipo tupla integra una serie de metodos integrados a continuacion:

I.- count()

Este metodo recibe un elemento como argumento, y cuenta la cantidad de veces que aparece en la tupla. 

                        >>> valores = ("Python", True, "Zope", 5)
                        >>> print "True ->", valores.count(True)
                        True -> 1
                        >>> print "'Zope' ->", valores.count('Zope')
                        'Zope' -> 1
                        >>> print "5 ->", valores.count(5)
                        5 -> 1

II.- index()

Comparte el mismo metodo index() de las listas. Este metodo recibe un elemento como argumento, y devuelve el indice de su primera aparicion en la tupla. 

                        >>> valores = ("Python", True, "Zope", 5)
                        >>> print valores.index(True)
                        1
                        >>> print valores.index(5)
                        3
