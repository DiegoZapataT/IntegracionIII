Bucle while

En Python existe una palabra reservada la cual nos ayudara a generar bucles, esta se llama while, nos permite ejecutar ciclos, o bien secuencias periódicas que nos permiten ejecutar código múltiples veces.

El ciclo while nos permite realizar multiples iteraciones basándonos en el resultado de una expresión lógica que puede tener como resultado un valor True o False.

Tipos de bucles while

Pueden existir variados tipos de ciclos while que se definen por su forma de estructurarse o bien por su forma de finalizar.

I.- Bucle while controlado por Conteo

Es un bucle while que se controla mediante un contador en este caso llamado numero:

              suma, numero = 0, 1
              while numero <= 10:
                  suma = numero + suma
                  numero = numero + 1
              print "La suma es " + str(suma)

En este ejemplo tiene un contador con valor inicial 0, cada iteración o vuelta del while manipula esta variable de manera que su valor incremente en 1, por lo que luego de su primera iteración el contador tendra un valor de 1, luego 2, y asi sucesivamente.

Eventualmente cuando el contador llegue a tener un valor de 10, la condicion del ciclo numero <= 10 sera False, por lo que el ciclo terminará arrojando el siguiente resultado.

II.- Bucle while controlado por Evento

Este bucle while tiene este nombre ya que terminara cuando un evento que estara dentro del mismo tenga un determinado resultado, un ejemplo de uso de este ciclo while es con la libreria pygame ya que al crear una ventan se requiere un ciclo while True, que luego de que se cierre o se produzca algo se ejecutara la instruccion break para que se rompa el ciclo. 

              promedio, total, contar = 0.0, 0, 0
              print "Introduzca la nota de un estudiante (-1 para salir): "
              grado = int(raw_input())	
              while grado != -1:
                  total = total + grado
                  contar = contar + 1
                  print "Introduzca la nota de un estudiante (-1 para salir): "
                  grado = int(raw_input())
              promedio = total / contar
              print "Promedio de notas del grado escolar es: " + str(promedio)

En este el evento que se dispara cuando el usuario ingresa el valor -1, causara que el bucle while se interrumpa o no se inicie.

III.- Bucle while con else

Al igual que la sentencia if, la estructura while tambien puede combinarse con una sentencia else

El nombre de la sentencia else es equivocada, ya que el bloque else se ejecutara en todos los casos, es decir, cuando la expresion condicional del while sea False. 

              promedio, total, contar = 0.0, 0, 0
              mensaje = "Introduzca la nota de un estudiante (-1 para salir): "
              
              grado = int(raw_input(mensaje))
              while grado != -1:
                  total = total + grado
                  contar += 1
                  grado = int(raw_input(mensaje))
              else:
                  promedio = total / contar
                  print "Promedio de notas del grado escolar: " + str(promedio)

La sentencia else tiene la ventaja de mantener el mismo nombre y la misma sintaxis que en las demás estructuras de control.

Sentencias utilitarias

Son sentencias que nos ayudaran de una u otra manera a terminar el ciclo while

I.- Sentencia break

El ciclo terminara mediante una interrupcion dentro del mismo mediante un break, aunque la condicion de finalizacion del mismo aun no se haya cumplido: 

              variable = 10

              while variable > 0:
                  print 'Actual valor de variable:', variable
                  variable = variable -1
                  if variable == 5:
                      break

II.- Sentencia continue

La sentencia continue hae que pase de nuevo al principio del bucle aunque no se haya terminado de ejecutar el ciclo anterior (Reinicia el ciclo al punto de inicio). 

              variable = 10

              while variable > 0:              
                 variable = variable -1
                 if variable == 5:
                    continue
                 print 'Actual valor de variable:', variable

Ejemplo

Sucesion de Fibonacci

Mediante el uso de un ciclo while se puede ejecutar y obtener todos los numeros de la sucesion de fibonacci desde el 0 al 100: 

              a, b = 0, 1
              while b < 100:
                  print b,
                  a, b = b, a + b
