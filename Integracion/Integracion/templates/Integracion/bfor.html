Bucle for

La sentencia for en Python difiere un poco de lo que uno puede estar acostumbrado, la sentencia for de python itera sobre los items de cualquier secuencia (generalmente una lista o una cadena de caracteres), en el orden que aparecen en la secuencia.

Tipos de Bucle for

Al igual que con el ciclo while, el ciclo for tiene variados tipos que, si bien son iguales, tienen unas pequeñas diferencias entre si

I.- Bucle for con Listas

El bucle for generalmente se usa para iterar listas, como podemos ver mas abajo:

                animales = ['gato', 'perro', 'serpiente']
                for animal in animales:
                    print "El animal es: {0}, tamaño de palabra es: {1}".format(
                        animal, len(animal))

II.- Bucle for con Listas y funcion range

El siguiente ejemplo nos muestra un for en que se usa la funcion range() la cual nos sirve para dar un valor final hasta el cual se quiera iterar. 

                oracion = 'Mary entiende muy bien Python'
                frases = oracion.split() # convierte a una lista cada palabra
                print "La oración analizada es:", oracion, ".\n"
                for palabra in range(len(frases)):
                    print "Palabra: {0}, en la frase su posición es: {1}".format(
                        frases[palabra], palabra)

Si se necesita iterar sobre una secuencia de números. Genera una lista conteniendo progresiones aritméticas, por ejemplo, como se hace en el fragmento que podemos observar arriba.

III.- Bucle for con Tuplas:

Itera de la misma manera de como lo hace sobre una lista o un string:

                conexion_bd = "127.0.0.1","root","123456","nomina"
                for parametro in conexion_bd:
                    print parametro

IV.- Bucle for con Diccionarios:

Un bucle for puede tambien iterar en una coleccion estructurada como clave+valor, de manera que se pueden obtener los valores que el diccionario contenga: 

            datos_basicos = {
                "nombres":"Leonardo Jose",
                "apellidos":"Caballero Garcia",
                "cedula":"26938401",
                "fecha_nacimiento":"03/12/1980",
                "lugar_nacimiento":"Maracaibo, Zulia, Venezuela",
                "nacionalidad":"Venezolana",
                "estado_civil":"Soltero"
            }
            clave = datos_basicos.keys()
            valor = datos_basicos.values()
            cantidad_datos = datos_basicos.items()
            
            for clave, valor in cantidad_datos:
                print clave + ": " + valor

Bucle for con else

Al igual que la sentencia if y el bucle while, la estructura for también puede combinarse con una sentencia else.

El nombre de la sentencia else es equivocada, ya que el bloque else se ejecutara en todos los casos, es decir, cuando la expresion condicional del bucle for sea False. 

            db_connection = "127.0.0.1","5432","root","nomina"
            for parametro in db_connection:
                print parametro
            else:
                print """El comando PostgreSQL es: 
            $ psql -h {server} -p {port} -U {user} -d {db_name}""".format(
                    server=db_connection[0], port=db_connection[1], 
                    user=db_connection[2], db_name=db_connection[3])
