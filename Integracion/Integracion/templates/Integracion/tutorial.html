    Tipos de datos

    Para comenzar este tutorial debemos aprender los tipos de datos que podemos encontrar en python, estos encerrados en 6 categorias distintas las cuales pueden ser mutable(su contenido o valor pueden cambiar en tiempo de ejecucion) inmutable(su contenido o valor no pueden cambiar en tiempo de ejecucion):

    1. Numeros inmutables:
      1. int: Entero
      2. long: Entero Long
      3. float: decimal
      4. complex: complejo
      5. bool: booleano
    2. Secuencias inmutables:
      1. str: cadena de caracteres
      2. unicode: cadena de caracteres Unicode
      3. tuple: tupla
      4. xrange: rango inmutable
    3. Secuencias mutables:
      1. list: lista
      2. range: rango mutable
    4. Mapeos:
      1. dict: diccionario
    5. Conjuntos Mutables:
      1. set: conjunto mutable
    6. Conjuntos inmutables:
      1. frozenset: conjunto inmutable
    1. Otros tipos de datos incorporados son los siguientes:
      1. NoneType: objeto none
      2. NotImplementedType: objeto no implementado
      3. ellipsis: Objeto Ellipsis
      4. file: objeto archivo/file

    Variables y constantes

    I.- Variables

    Es un nombre que se refiere a un objeto que reside en la memoria. El objeto puede ser de alguno de los tipos vistos mas arriba (numero o cadena de texto), o alguno de los otros tipos existentes en python.

    Cada variable debe tener un nombre unico llamado identificador. Eso es muy de pensar las variables como contenedores que tienen la data el cual puede ser cambiado despues a través de un proceso de programacion

    II.- Alcance de variables

    Las variables en Python son locales por defecto. Esto quiere decir que las variables definidas y utilizadas en el codigo de una funcion, solo viven y tienen existencia dentro de esta, y a su vez no interfieren con otras variables del resto del codigo.

    Ejemplos de variables:

    A continuación veremos ejemplos de uso de variables:

    Ejemplo de asignacion de valor a una variable

    A continuacion, se creara un par de variables como ejemplos. Una de tipo str (Cadena de caracteres) y una de tipo int (entero):

                      >>>
                  c="Hola Mundo"
                  # cadenas de caracteres
                  >>> type(c) # comprobar tipo de dato
                  <type 'str'>
                  >>> e = 23 # número entero
                  >>> type(e) # comprobar tipo de dato
                  <type 'int'>

    Como podemos observar arriba, la asignacion de variables en python es mucho mas comoda y amigable, ya que por ejemplo, en java se debe anteponer el tipo de dato que la variable contendra: 

                      String c = "Hola Mundo";
                  int e = 23;

    Para que no existe confusion, lo que se encuentra escrito despues de un # pertenece a un comentario, fragmentos de codigo que el programa no tomara en cuenta, los cuales nos sirven para darle entendimiento a lo que hace nuestro codigo.

    Ejemplo de asignar múltiples valores a múltiples variables

    A continuacion, se creará multiples variables (int, float, str) asignando multiples valores:

                      >>> a, b, c = 5, 3.2, "Hola"
                  >>> print a
                  5
                  >>> print b
                  3.2
                  >>> print c
                  'Hola'

    III.- Constantes

    Una constante es un tipo de variable en python que no puede o no debe ser cambiada. Eso es muy de ayuda pensar las constantes como contenedores que contienen informacion la cual no puede ser cambiada despues.

    En Python, las constantes son usualmente declaradas y asignadas en un modulo.

    Ejemplo de constantes desde un modulo externo:

    Creamos un archivo llamado constantes.py con el contenido siguiente:

                      IP_DB_SERVER = "127.0.0.1"
                  PORT_DB_SERVER = 3307
                  USER_DB_SERVER = "root"
                  PASSWORD_DB_SERVER = "123456"
                  DB_NAME = "nomina"

    Crear un archivo llamado main.py con el siguiente contenido:

                      import constantes
                  print "scp -v -P {0} {1}@{2}:/{3}/{4}/{4}.sql /srv/backup".format(
                  str(constantes.PORT_DB_SERVER), constantes.USER_DB_SERVER, 
                  constantes.IP_DB_SERVER, constantes.USER_DB_SERVER,
                  constantes.DB_NAME)

    Luego ejecuta el programa escribiendo en el cmd dentro de la carpeta del programa python main.py deberiamos tener la siguiente salida: 

    scp -v -P 3307 root@127.0.0.1:/root/webapp/db.sql /srv/backup

    IV.- Operadores

    En Python existen operadores los cuales nos ayudaran a realizar distintas operaciones aritmetica/logicas, los operadores que mas usaremos durante la programacion son los siguientes:

    1. Operadores Aritmeticos:
      1. Operador de suma "+", el cual nos servira para sumar datos numericos o concatenar strings
      2. Operador de resta "-", el cual nos servira para restar valores numericos
      3. Operador de negacion "-x", el cual nos servira para asignar valor negativo a un numero x
      4. Operador de multiplicacion "*", el cual nos servira para multiplicar sumar dos o mas numeros
      5. Operador de exponente "**", el cual nos servira para trabar con potencias para asignar un exponente a un numero
      6. Operador de division "/", el cual nos servira para realizar la division de valores numericos con resultado entero o decimal
      7. Operador de division entera "//", el cual realiza la misma operacion que el anterior pero solo entrega resultados enteros, si nuestro resultado es 1,75 nos dara como resultado un 1.0
    2. Operadores Relacionales:
      1. Operador ==: Evaluara que los valores sean iguales para varios tipos de datos.
        Por Ejemplo: a == b si ambos son un 1 nos devolvara True, caso contrario si fueran un 1 y un 2 nos devolvera False.
      2. Operador !=: Evaluara que los valores sean distintos, todo lo contrario al nombrado anteriormente.
      3. Operador <: Evaluara si el valor del lado izquierdo es menor que el valor del lado derecho.
      4. Operador >: Evaluara si el valor del lado izquierdo es mayor que el valor del lado derecho.
      5. Operador <=: Evaluara si el valor del lado izquierdo es menor o igual que el valor del lado derecho.
      6. Operador >=: Evaluara si el valor del lado izquierdo es mayor o igual que el valor del lado derecho.
